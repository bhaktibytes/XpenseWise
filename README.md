## 📌 What is XpenseWise?

**XpenseWise** is an AI-powered gamified finance platform designed to help students manage money smarter through behavioural analytics, personalized insights, and interactive financial experiences.

It combines **AI, gamification, behavioural psychology, and fintech** to transform traditional expense tracking into an engaging and motivating financial companion.

> *"Not just expense tracking — a financial companion that grows with you."*

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🧠 AI Financial Mentor | Personalized spending insights & smart saving suggestions |
| 😊 Mood-based Tracking | Expense tracking with emotional spending analysis |
| 🎮 Gamified Finance | XP levels, streaks, badges & saving challenges |
| 📊 Analytics Dashboard | Spending trends, budget tracking & interactive charts |
| 📸 Smart Receipts | Image preview & transaction organization |
| 💹 Investment Simulator | Micro-investment simulator with compound interest forecasting |
| ⚡ Performance | Debounced search, lazy loading & memoized filtering |
| 🎨 Modern UI | Mobile-first fintech UI with glassmorphism design & animations |
